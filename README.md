# 던전위딘 위키 (Dungeon Within Wiki)

> **"Reclaim Your Throne as the True Dark Lord!"**

GameCoaster의 로그라이크 던전 디펜스 게임 **던전위딘 (Dungeon Within)** 의 정보 위키입니다.

🔗 **사이트**: https://choyeun.github.io/dungeon-within/

## 개요

어둠의 군주가 되어 무한한 꿈의 세계에서 던전을 건설하고, 몬스터를 모으며, 함정을 설치해 침략자들을 막아내는 로그라이크 게임.

- **개발사**: GameCoaster
- **플랫폼**: Android
- **가격**: ₩9,500 (유료)
- **장르**: Roguelike / 던전 디펜스 / 싱글플레이어
